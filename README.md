# DNA Educational Games

A Spring Boot application featuring two interactive biology games:
1.  **DNA Replication**: Learn base pairing and replication.
2.  **Protein Synthesis**: Learn transcription (DNA->mRNA) and translation (mRNA->Protein).

## Prerequisites
- Java 17 or higher
- Maven

## How to Run
1.  Open a terminal in this directory.
2.  Run the application:
    ```bash
    mvn spring-boot:run
    ```
3.  Open your browser to [http://localhost:8080](http://localhost:8080).

## Features
- **Interactive Drag-and-Drop**: Engaging gameplay for learning nucleotide pairing.
- **Visual Feedback**: Immediate validation of correct/incorrect pairs with animations.
- **Scientific Accuracy**: Follows standard base pairing rules (A-T, C-G, A-U).
- **Responsive Design**: Works on desktop and tablet sizes.
